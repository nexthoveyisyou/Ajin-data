Big Data

빅데이터 엔지니어
 - 여러 대의 컴퓨터를 운용할 수 있는 시스템 구축
 - 데이터 베이스 구축, 설계
 - 센서로부터 데이터 저장까지의 프로세스 설계

빅데이터 분석가 - analyst
 - 모아진 빅데이터를 활용하여 -> 경영자에게 의사결정을 위한 근거를 제시
 - pandas, matplotlib 등 분석 툴 적극적으로 활용
 - 비전문가를 대상으로 프레젠테이션 (레포트 작성)
 - 머신러닝은 보조 수단으로 활용
 - 통계/확률 이론을 적극적으로 활용
 - tabular 데이터 (테이블 형식, 시계열 데이터)
 
빅데이터 사이언티스트 - scientist
 - 인공지능/머신러닝을 주력으로 하는 사람들
 - 현상에 대한 모델을 생성 
   (예 : GDP와 행복 간의 상관관계, 지니계수와 정치적 자유의 상관관계)
   (비정형 : 이미지, 음성, 동영상 등.. 다양한 데이터를 다룸)
   
   딥러닝/인공지능 -> 데이터를 대량으로 필요로 함
   빅데이터 -> 쌓인 데이터를 분석할 인공지능이 필요 

빅데이터 디렉터
 - 전체 시스템, 운영을 계획, 실현하는 사람
   (클라이언트의 요구사항, 필요한 자원 - 하드웨어, 소프트웨어, 인력, 예산, 시간... ) -> 클라이언트-개발자 연결, 요구사항들을 점검
